### 核心功能
- ✨ 书签管理（增删改查）
- 📁 多层级文件夹支持
- 🔍 实时智能搜索
- 📤 导入导出（支持主流浏览器）
- 💾 本地数据存储（IndexedDB）
- 🎨 现代化 UI 设计

### 技术栈
- **前端框架**: Vue 3.5.19 (Composition API)
- **开发语言**: TypeScript 5.9.2
- **构建工具**: Vite 7.1.3
- **UI 组件**: Element Plus 2.10.7
- **数据存储**: Dexie.js (IndexedDB)
- **代码格式**: Prettier
